# Personal Resume Website

A responsive and accessible personal resume website created for an IT 161 assignment.

The website presents my education, technical skills, academic and personal IT experience, and certifications. It was developed using semantic HTML5 and modern CSS.

## Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Git
- GitHub
- GitHub Pages

## Accessibility Features

- Semantic HTML elements
- Sequential heading hierarchy
- Keyboard-accessible navigation
- Visible keyboard focus indicators
- Skip-to-content link
- Descriptive link text
- Responsive text and layouts
- Reduced-motion support
- Screen-reader-only context for links that open new tabs
- Contact information marked with an `address` element
- Dates marked with `time` elements
