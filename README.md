# Personal Resume Website

## Overview

This project is a responsive personal resume website created for my IT 161 coursework. It showcases my education, technical skills, certifications, and hands-on IT experience using modern semantic HTML5 and CSS3.

The website was designed with accessibility and responsive web design in mind, following semantic HTML best practices and WCAG accessibility principles.

## Features

- Responsive layout for desktop and mobile devices
- Semantic HTML5 structure
- Accessible navigation
- CSS Grid and Flexbox layout
- Professional resume presentation
- Keyboard-friendly navigation
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- CSS Grid
- CSS Flexbox
- Git
- GitHub
- GitHub Pages

## Running the Project

1. Clone the repository:

```bash
git clone https://github.com/IT-161/personal-resume.git
```

2. Open the project folder.

3. Open `index.html` in a web browser, or launch it using the VS Code Live Server extension.
